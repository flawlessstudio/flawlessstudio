# Audits — voice-ai

> Periodic audit records for the `voice-ai` registry.

Each audit documents the state of `registry.core.yaml` and `registry.watchlist.yaml` at a point in time: entries reviewed, promotions made, removals justified, and next review date.

## Naming convention

```
YYYY-MM-DD-voice-ai-audit.md
```

## Audit scope

- Core entries: still valid, correctly categorized, no duplicates
- Watchlist entries: promotion criteria met or not; `promote_after` date respected
- Schema compliance: both YAML files validate against current schema
