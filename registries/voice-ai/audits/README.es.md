# Auditorías — voice-ai

> Registros de auditoría periódica del registro `voice-ai`.

Cada auditoría documenta el estado de `registry.core.yaml` y `registry.watchlist.yaml` en un momento dado: entradas revisadas, promociones realizadas, eliminaciones justificadas y fecha de próxima revisión.

## Convención de nombres

```
YYYY-MM-DD-voice-ai-audit.md
```

## Alcance de la auditoría

- Entradas core: siguen siendo válidas, correctamente categorizadas, sin duplicados
- Entradas watchlist: criterios de promoción cumplidos o no; fecha `promote_after` respetada
- Conformidad con el esquema: ambos archivos YAML validan contra el esquema actual
