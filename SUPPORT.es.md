# Soporte

## Cómo obtener ayuda

Este repositorio forma parte del ecosistema Flawless Studio — un proyecto de documentación y estándares mantenido por un único autor.

### Para preguntas sobre el canon o los estándares

Abre una [GitHub Discussion](https://github.com/flawlessstudio/flawlessstudio/discussions) con la etiqueta `question`.

### Para errores o problemas en la documentación

Abre un [issue](https://github.com/flawlessstudio/flawlessstudio/issues) usando la plantilla correspondiente.

### Para contacto directo

Email: `oneflawlessstudio@gmail.com`

## Tiempo de respuesta

Este proyecto lo mantiene una sola persona. Las respuestas son de mejor esfuerzo, normalmente en un máximo de 5 días hábiles.

## Lo que este proyecto no ofrece

- Soporte en tiempo real ni chat
- Consultoría personalizada a través de issues
- Solicitudes de funcionalidades sin discusión previa
