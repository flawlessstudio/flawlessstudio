# Assets

> Assets visuales y de marca estáticos del ecosistema Flawless Studio.

## Estructura

```
assets/
└── profile/
    ├── flawless-avatar.png   — avatar de perfil (ráster, aprox. 512×512px)
    ├── flawless-avatar.svg   — avatar de perfil (vectorial, escalable)
    └── flawless-banner.svg   — banner de perfil de GitHub (vectorial, escalable)
```

## Uso

- Usa las variantes SVG siempre que sea posible para independencia de resolución.
- La variante PNG se proporciona para plataformas que no soportan SVG.
- No modifiques los assets de marca sin una revisión formal aprobada vía `GOVERNANCE.md`.
