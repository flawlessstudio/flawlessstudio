# Assets

> Static brand and visual assets for the Flawless Studio ecosystem.

## Structure

```
assets/
└── profile/
    ├── flawless-avatar.png   — profile avatar (raster, 512×512px approx.)
    ├── flawless-avatar.svg   — profile avatar (vector, scalable)
    └── flawless-banner.svg   — GitHub profile banner (vector, scalable)
```

## Usage

- Use SVG variants wherever possible for resolution independence.
- PNG variant is provided for platforms that do not support SVG.
- Do not modify brand assets without a formal revision approved via `GOVERNANCE.md`.
