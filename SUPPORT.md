# Support

## How to get help

This repository is part of the Flawless Studio ecosystem — a documentation and standards project maintained by a solo practitioner.

### For questions about the canon or standards

Open a [GitHub Discussion](https://github.com/flawlessstudio/flawlessstudio/discussions) with the `question` label.

### For bugs or errors in documentation

Open an [issue](https://github.com/flawlessstudio/flawlessstudio/issues) using the appropriate template.

### For direct contact

Email: `oneflawlessstudio@gmail.com`

## Response time

This project is maintained by one person. Responses are best-effort, typically within 5 business days.

## What this project does not support

- Real-time support or chat
- Custom consulting engagements via GitHub issues
- Feature requests without prior discussion

If you have identified the right channel for your case, this document has done its job.
