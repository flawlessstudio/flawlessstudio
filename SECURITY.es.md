# Política de seguridad

## Versiones con soporte

Este repositorio contiene documentación y estándares, no código ejecutable. No existen versiones con parches de seguridad activos.

## Cómo reportar una vulnerabilidad

Si descubres un problema de seguridad en cualquier repositorio o artefacto del ecosistema Flawless Studio:

1. **No abras un issue público.**
2. Envía un email a `oneflawlessstudio@gmail.com` con el asunto `[SECURITY] <descripción breve>`.
3. Incluye: repositorio afectado, descripción del problema y pasos de reproducción si aplica.
4. Recibirás respuesta en un máximo de 72 horas.

## Alcance

- Credenciales o secretos comprometidos accidentalmente
- Contenido malicioso introducido mediante pull request
- Riesgos en la cadena de suministro del tooling referenciado

## Fuera de alcance

- Vulnerabilidades teóricas sin impacto demostrable
- Problemas en herramientas de terceros no mantenidas por Flawless Studio
