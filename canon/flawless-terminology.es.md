# Terminología Flawless

> Glosario canónico de todos los términos utilizados en el ecosistema Flawless. En caso de duda, consultar aquí.

## Términos Principales

| Término | Definición |
|---|---|
| **Canon** | El conjunto inmutable de documentos que definen el sistema Flawless |
| **Ecosistema** | La red completa de repositorios, herramientas y agentes Flawless |
| **Marco** | Los principios operativos y reglas estructurales de Flawless Studio |
| **Filtro Flawless** | La barrera de calidad aplicada a cada decisión, artefacto y sistema |
| **INUI OS** | El sistema operativo de diseño que impulsa la UI Flawless |
| **FDIS OS** | Flawless Design Intelligence System — capa de diseño nativa de IA |
| **Flawless Engine** | Capa de automatización y orquestación principal |
| **Registro** | Un catálogo versionado de herramientas, stacks o agentes activos |
| **Runbook** | Guía operativa paso a paso para una tarea o sistema específico |
| **Contrato de Agente** | Especificación formal del alcance, entradas, salidas y restricciones de un agente IA |
| **Diátaxis** | Marco de documentación que separa tutoriales, how-to, explicación y referencia |
| **Par bilingüe** | Un documento canónico y su traducción ES (`file.md` + `file.es.md`) |
