# Tutoriales

> Recorridos orientados al aprendizaje del ecosistema Flawless Studio.

Los tutoriales guían a un nuevo colaborador a través de una tarea completa y significativa de principio a fin. Priorizan la comprensión sobre la eficiencia y no asumen conocimiento previo del sistema.

## Ámbito

- Primeros pasos en el ecosistema Flawless
- Tu primera contribución al canon
- Entender la estructura de registros de principio a fin
- Cómo funciona el sistema de paridad bilingüe

*Para instrucciones específicas de tareas, ver [`docs/how-to/`](../how-to/).*
