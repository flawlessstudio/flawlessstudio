# Runbooks

> Procedimientos operativos paso a paso para mantener el ecosistema Flawless Studio.

Los runbooks están escritos para el mantenedor. Cubren tareas operativas repetibles que deben ejecutarse correctamente y en orden. Cada runbook incluye precondiciones, pasos, validación y rollback si aplica.

## Ámbito

- Proceso de revisión de documentos del canon
- Procedimiento de auditoría de registros
- Verificación del estado del repositorio
- Verificación de paridad bilingüe
- Procedimiento de actualización del mapa del ecosistema
