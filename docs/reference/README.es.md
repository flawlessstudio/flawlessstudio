# Referencia

> Material de consulta autoritativo del ecosistema Flawless Studio.

Los documentos de referencia están optimizados para escanear, no para leer. Contienen los datos necesarios para usar el sistema correctamente: esquemas, definiciones de campos, valores de estado, convenciones de nombres y listas canónicas.

## Ámbito

- Definiciones de campos de documentos del canon
- Referencia de esquema de registros
- Convenciones de nombres en todo el ecosistema
- Definiciones de valores de estado
- Mapa de enlaces entre repositorios
