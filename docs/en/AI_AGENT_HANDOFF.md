# AI Agent Handoff

## Objective

Use this file when an AI agent needs to understand, extend or apply INUI without drifting away from the canon.

## System identity

- Positioning: Editorial Minimal Tech Premium
- Primary mode: Light
- Secondary mode: Dark
- Brand center: Ku Crimson `#E70F0E`

## Ranked territories

1. Brand / Editorial Studio Core
2. Web / Case Studies / Presentation
3. Product UI / Functional Interface
4. Documentation / Slides / Assets
5. Dark Mode as a complementary expression

## Hard rules

1. Design for light mode first. Dark mode is derived, not independent.
2. Do not use more than one dominant warm accent within the same local composition.
3. Do not let components become more expressive than the overall layout and type system.
4. Do not use moodboards or product mockups as direct UI laws without translation.
5. Do not add product-specific components unless the product scope is explicit.
6. When in doubt, preserve the core grammar instead of adding novelty.
