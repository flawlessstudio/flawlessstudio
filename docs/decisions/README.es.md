# Decisiones

> Registros de decisiones de arquitectura (ADR) del ecosistema Flawless Studio.

Cada archivo documenta una única decisión: el contexto, las opciones consideradas, la elección tomada y el razonamiento. Los ADR son inmutables una vez aceptados — las decisiones superadas se marcan como tal, nunca se eliminan.

## Convención de nombres

```
ADR-NNNN-<slug>.md
```

Ejemplo: `ADR-0002-registry-schema.md`

## Valores de estado

- `proposed` — en discusión
- `accepted` — en vigor
- `superseded-by: ADR-XXXX` — reemplazado por una decisión más reciente
- `rejected` — considerado y rechazado
