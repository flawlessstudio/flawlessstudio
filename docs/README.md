# Docs

> Operational knowledge base for Flawless Studio. Organized by the [Diátaxis](https://diataxis.fr) framework.

## Structure

| Folder | Type | Purpose |
|---|---|---|
| `tutorials/` | Learning-oriented | Step-by-step guides for new systems or tools |
| `how-to/` | Task-oriented | Concrete instructions for specific tasks |
| `explanation/` | Understanding-oriented | Conceptual background and rationale |
| `reference/` | Information-oriented | Agent contracts, API specs, schemas |
| `decisions/` | Record-oriented | Architecture decisions and migration logs |
| `runbooks/` | Operations-oriented | Operational guides, AI knowledge docs |
| `audits/` | Quality-oriented | Audit reports and quality snapshots |

## Language Convention

- `filename.md` — English (authoritative)
- `filename.es.md` — Spanish (localized)
