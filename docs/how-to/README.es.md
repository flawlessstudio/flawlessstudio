# Guías de procedimiento

> Instrucciones orientadas a tareas para trabajar dentro del ecosistema Flawless Studio.

Las guías de procedimiento responden a *cómo hago X*. Asumen que el lector tiene contexto y quiere completar una tarea concreta de forma eficiente.

## Ámbito

- Cómo añadir un nuevo documento de canon
- Cómo promover una entrada de registro de watchlist a core
- Cómo abrir una revisión de canon mediante el proceso de gobernanza
- Cómo contribuir un nuevo vertical de registro
